# Loops

This section covers loops in Python, including `for` and `while`.

---

## Key Concepts
- **`for` Loops**: Iterating over sequences.
- **`while` Loops**: Repeating code while a condition is true.
- **Loop Control**: `break`, `continue`, `pass`
