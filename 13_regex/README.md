# Regular Expressions

This section covers how to use regular expressions in Python.

---

## Key Concepts
- **Pattern Matching**: Using `re.match()` and `re.search()`.
- **Substitution**: Replacing text with `re.sub()`.
- **Splitting Strings**: `re.split()`.
