# Задания по теме "Регулярные выражения"

1. Напиши программу, которая ищет в тексте все email-адреса.
2. Напиши программу, которая заменяет все цифры в тексте на символ `*`.
3. Напиши программу, которая проверяет, является ли строка валидным номером телефона.
4. Напиши программу, которая ищет все слова, начинающиеся с буквы "А".
5. Напиши программу, которая разделяет текст на предложения.
