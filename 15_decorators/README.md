# Decorators

This section covers how to create and use decorators in Python.

---

## Key Concepts
- **Function Decorators**: Modifying functions.
- **Class Decorators**: Modifying classes.
- **Built-in Decorators**: `@staticmethod`, `@classmethod`, `@property`.
